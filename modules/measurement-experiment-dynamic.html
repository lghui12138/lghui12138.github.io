实验与测量基本概念

实验与测量是流体力学研究的重要手段，包括压力测量、速度测量、流量测量、温度测量等。准确的测量技术是验证理论和指导工程实践的基础。

压力测量
实验与测量
$$p = \rho gh$$
静水压力测量原理
推导过程：
静水压力：$p = \rho gh$
U型管压力计：$\Delta p = \rho g \Delta h$
差压计：$p_1 - p_2 = \rho g(h_1 - h_2)$
倾斜管压力计：$p = \rho g L \sin \theta$
其中L为倾斜管长度，θ为倾斜角
物理意义：
压力测量是流体力学实验的基础。通过测量液柱高度可以确定压力大小，这是最简单可靠的测量方法。U型管、倾斜管等是常用的压力测量装置。

皮托管
实验与测量
$$v = \sqrt{\frac{2(p_0 - p)}{\rho}}$$
皮托管测速公式
推导过程：
伯努利方程：$\frac{p}{\rho} + \frac{v^2}{2} = \frac{p_0}{\rho}$
总压：$p_0 = p + \frac{1}{2}\rho v^2$
动压：$p_0 - p = \frac{1}{2}\rho v^2$
速度：$v = \sqrt{\frac{2(p_0 - p)}{\rho}}$
皮托管系数：$v = C\sqrt{\frac{2(p_0 - p)}{\rho}}$
物理意义：
皮托管是最常用的流速测量仪器，通过测量总压和静压的差值来确定流速。皮托管系数C考虑了实际流动与理想流动的差异。

文丘里管
实验与测量
$$Q = C_d A_2 \sqrt{\frac{2(p_1 - p_2)}{\rho(1 - \beta^4)}}$$
文丘里管流量测量公式
推导过程：
连续性方程：$A_1 v_1 = A_2 v_2$
伯努利方程：$\frac{p_1}{\rho} + \frac{v_1^2}{2} = \frac{p_2}{\rho} + \frac{v_2^2}{2}$
面积比：$\beta = \frac{A_2}{A_1}$
速度：$v_2 = \sqrt{\frac{2(p_1 - p_2)}{\rho(1 - \beta^4)}}$
流量：$Q = C_d A_2 v_2$
物理意义：
文丘里管是常用的流量测量装置，通过收缩段产生压力差来测量流量。流量系数Cd考虑了实际流动中的能量损失。

热线风速仪
实验与测量
$$Nu = A + B Re^n$$
热线风速仪标定公式
推导过程：
努塞尔数：$Nu = \frac{hD}{k}$
雷诺数：$Re = \frac{\rho vD}{\mu}$
传热系数：$h = \frac{q}{A(T_w - T_f)}$
热线功率：$P = I^2R$
标定关系：$Nu = A + B Re^n$
物理意义：
热线风速仪通过测量热线与流体之间的传热来确定流速。热线温度随流速变化，通过标定可以得到流速与温度的关系。

激光多普勒测速
实验与测量
$$f_D = \frac{2v \sin \frac{\theta}{2}}{\lambda}$$
多普勒频移公式
推导过程：
多普勒效应：$f_D = f_s - f_0$
散射光频率：$f_s = f_0(1 + \frac{2v \cos \alpha}{c})$
多普勒频移：$f_D = \frac{2v \cos \alpha}{\lambda}$
双光束系统：$f_D = \frac{2v \sin \frac{\theta}{2}}{\lambda}$
其中θ为两光束夹角
物理意义：
激光多普勒测速仪利用多普勒效应测量流体速度。通过测量散射光的多普勒频移可以精确确定流速，是非接触式测量的重要方法。

粒子图像测速
实验与测量
$$\vec{v} = \frac{\Delta \vec{x}}{\Delta t}$$
PIV速度计算原理
推导过程：
粒子位移：$\Delta \vec{x} = \vec{x}_2 - \vec{x}_1$
时间间隔：$\Delta t = t_2 - t_1$
速度：$\vec{v} = \frac{\Delta \vec{x}}{\Delta t}$
互相关函数：$R(\vec{s}) = \int I_1(\vec{x}) I_2(\vec{x} + \vec{s}) d\vec{x}$
峰值位置：$\vec{s}_{max} = \arg \max R(\vec{s})$
物理意义：
PIV通过追踪示踪粒子的运动来测量流场速度。通过分析两帧图像中粒子的位移，可以计算整个流场的速度分布。

实际应用例题
实验与测量
皮托管测速计算
使用皮托管测量管道中的流速
解题过程：
已知：$\Delta p = 1000 \text{ Pa}, \rho = 1.225 \text{ kg/m}^3, C = 0.98$
皮托管公式：$v = C\sqrt{\frac{2\Delta p}{\rho}}$
$v = 0.98\sqrt{\frac{2 \times 1000}{1.225}}$
$v = 0.98 \times 40.4 = 39.6 \text{ m/s}$
测量误差：$\delta v = \pm 2\% \times 39.6 = \pm 0.8 \text{ m/s}$
结果分析：
管道中的流速为39.6 m/s，测量误差为±0.8 m/s。皮托管是测量流速的可靠方法，适用于各种流动条件。
