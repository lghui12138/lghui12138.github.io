边界层基本概念

边界层理论是流体力学中的重要概念，描述了固体壁面附近粘性效应起主导作用的薄层区域。边界层内速度梯度很大，粘性力与惯性力相当。

边界层厚度定义
边界层理论
$$\delta = \int_0^\infty \left(1 - \frac{u}{U_\infty}\right) dy$$
位移厚度，表示边界层对外部流动的影响
推导过程：
边界层厚度定义为速度达到外部流动速度99%的位置
位移厚度：$\delta^* = \int_0^\infty \left(1 - \frac{u}{U_\infty}\right) dy$
动量厚度：$\theta = \int_0^\infty \frac{u}{U_\infty}\left(1 - \frac{u}{U_\infty}\right) dy$
能量厚度：$\delta^{**} = \int_0^\infty \frac{u}{U_\infty}\left(1 - \frac{u^2}{U_\infty^2}\right) dy$
这些厚度参数描述了边界层对外部流动的影响
物理意义：
边界层厚度描述了粘性效应影响的范围。位移厚度表示边界层对外部流动的"排挤"效应，动量厚度表示动量损失，能量厚度表示能量损失。

平板边界层方程
边界层理论
$$u\frac{\partial u}{\partial x} + v\frac{\partial u}{\partial y} = \nu\frac{\partial^2 u}{\partial y^2}$$
平板边界层的简化动量方程
推导过程：
从Navier-Stokes方程出发
边界层内：$\frac{\partial}{\partial x} \ll \frac{\partial}{\partial y}$
压力梯度：$\frac{\partial p}{\partial x} = 0$（平板）
连续性方程：$\frac{\partial u}{\partial x} + \frac{\partial v}{\partial y} = 0$
动量方程：$u\frac{\partial u}{\partial x} + v\frac{\partial u}{\partial y} = \nu\frac{\partial^2 u}{\partial y^2}$
物理意义：
平板边界层方程是边界层理论的基础，描述了边界层内速度分布的变化规律。通过相似性解可以得到Blasius解。
边界层理论 平板边界层 动量方程
Blasius解
边界层理论
$$f''' + \frac{1}{2}ff'' = 0$$
Blasius方程，平板边界层的相似性解
推导过程：
引入相似性变量：$\eta = y\sqrt{\frac{U_\infty}{\nu x}}$
流函数：$\psi = \sqrt{\nu U_\infty x} f(\eta)$
速度分量：$u = U_\infty f'(\eta), v = \frac{1}{2}\sqrt{\frac{\nu U_\infty}{x}}(\eta f' - f)$
代入边界层方程得到：$f''' + \frac{1}{2}ff'' = 0$
边界条件：$f(0) = f'(0) = 0, f'(\infty) = 1$
物理意义：
Blasius解是平板边界层的精确解，通过相似性变换将偏微分方程转化为常微分方程。解得的流函数描述了边界层内的速度分布。

边界层厚度公式
边界层理论
$$\delta = 5.0\sqrt{\frac{\nu x}{U_\infty}}$$
平板边界层厚度随距离的平方根增长
推导过程：
从Blasius解得到：$\delta = 5.0\sqrt{\frac{\nu x}{U_\infty}}$
位移厚度：$\delta^* = 1.72\sqrt{\frac{\nu x}{U_\infty}}$
动量厚度：$\theta = 0.664\sqrt{\frac{\nu x}{U_\infty}}$
形状因子：$H = \frac{\delta^*}{\theta} = 2.59$
壁面剪切应力：$\tau_w = 0.332\rho U_\infty^2\sqrt{\frac{\nu}{U_\infty x}}$
物理意义：
边界层厚度随距离的平方根增长，这是边界层理论的重要特征。壁面剪切应力随距离的平方根减小，反映了边界层发展的规律。

压力梯度边界层
边界层理论
$$u\frac{\partial u}{\partial x} + v\frac{\partial u}{\partial y} = U_e\frac{dU_e}{dx} + \nu\frac{\partial^2 u}{\partial y^2}$$
考虑压力梯度的边界层方程
推导过程：
从Navier-Stokes方程出发
边界层近似：$\frac{\partial}{\partial x} \ll \frac{\partial}{\partial y}$
外部流动：$U_e\frac{dU_e}{dx} = -\frac{1}{\rho}\frac{\partial p}{\partial x}$
边界层方程：$u\frac{\partial u}{\partial x} + v\frac{\partial u}{\partial y} = U_e\frac{dU_e}{dx} + \nu\frac{\partial^2 u}{\partial y^2}$
连续性方程：$\frac{\partial u}{\partial x} + \frac{\partial v}{\partial y} = 0$
压力梯度影响边界层发展
物理意义：
压力梯度边界层方程考虑了外部流动对边界层的影响。正压力梯度（逆压梯度）会导致边界层增厚，甚至分离；负压力梯度（顺压梯度）会使边界层变薄。

Falkner-Skan相似性解
边界层理论
$$f''' + \frac{m+1}{2}ff'' + m(1-f'^2) = 0$$
Falkner-Skan方程，考虑压力梯度的相似性解
推导过程：
外部速度：$U_e(x) = U_0x^m$
相似性变量：$\eta = y\sqrt{\frac{(m+1)U_e}{2\nu x}}$
流函数：$\psi = \sqrt{\frac{2\nu U_e x}{m+1}} f(\eta)$
速度分量：$u = U_e f'(\eta), v = -\sqrt{\frac{\nu U_e}{2x(m+1)}}[(m+1)f + (m-1)\eta f']$
代入边界层方程得到：$f''' + \frac{m+1}{2}ff'' + m(1-f'^2) = 0$
边界条件：$f(0) = f'(0) = 0, f'(\infty) = 1$
物理意义：
Falkner-Skan解是考虑压力梯度的边界层相似性解。参数m控制压力梯度：m>0为顺压梯度，m<0为逆压梯度，m=0为平板边界层。这种解可以描述楔形流动和圆锥流动。
边界层理论 Falkner-Skan 相似性解
边界层分离条件
边界层理论
$$\left.\frac{\partial u}{\partial y}\right|_{y=0} = 0$$
壁面剪切应力为零的分离条件
推导过程：
壁面剪切应力：$\tau_w = \mu\left.\frac{\partial u}{\partial y}\right|_{y=0}$
分离条件：$\tau_w = 0$
即：$\left.\frac{\partial u}{\partial y}\right|_{y=0} = 0$
从边界层方程：$u\frac{\partial u}{\partial x} + v\frac{\partial u}{\partial y} = U_e\frac{dU_e}{dx} + \nu\frac{\partial^2 u}{\partial y^2}$
在壁面：$v = 0, u = 0$
得到：$\left.\frac{\partial^2 u}{\partial y^2}\right|_{y=0} = \frac{U_e}{\nu}\frac{dU_e}{dx}$
物理意义：
边界层分离发生在壁面剪切应力为零的位置。分离通常由逆压梯度引起，导致流动从壁面分离，形成回流区。分离点后流动变得复杂，需要特殊处理。
边界层理论 边界层分离 逆压梯度
边界层积分方程
边界层理论
$$\frac{d\theta}{dx} + (2+H)\frac{\theta}{U_e}\frac{dU_e}{dx} = \frac{C_f}{2}$$
von Kármán积分方程
推导过程：
动量积分：$\int_0^\delta \rho u\frac{\partial u}{\partial x} dy + \int_0^\delta \rho v\frac{\partial u}{\partial y} dy = -\delta\frac{\partial p}{\partial x} - \tau_w$
利用连续性方程：$\frac{\partial v}{\partial y} = -\frac{\partial u}{\partial x}$
积分得到：$\frac{d}{dx}\int_0^\delta \rho u(U_e - u) dy = \tau_w$
定义动量厚度：$\theta = \int_0^\delta \frac{u}{U_e}(1-\frac{u}{U_e}) dy$
von Kármán方程：$\frac{d\theta}{dx} + (2+H)\frac{\theta}{U_e}\frac{dU_e}{dx} = \frac{C_f}{2}$
其中：$H = \frac{\delta^*}{\theta}$为形状因子
物理意义：
von Kármán积分方程是边界层理论的重要工具，通过积分方法避免了求解偏微分方程的复杂性。形状因子H描述了边界层速度剖面的形状，是判断边界层状态的重要参数。
边界层理论 积分方程 von Kármán
湍流边界层
边界层理论
$$\frac{u}{u_\tau} = \frac{1}{\kappa}\ln\frac{yu_\tau}{\nu} + B$$
湍流边界层对数律
推导过程：
摩擦速度：$u_\tau = \sqrt{\frac{\tau_w}{\rho}}$
壁面律：$\frac{u}{u_\tau} = f(\frac{yu_\tau}{\nu})$
对数律：$\frac{u}{u_\tau} = \frac{1}{\kappa}\ln\frac{yu_\tau}{\nu} + B$
卡门常数：$\kappa \approx 0.41$
常数：$B \approx 5.0$
适用条件：$30 < \frac{yu_\tau}{\nu} < 0.1\frac{\delta u_\tau}{\nu}$
物理意义：
湍流边界层对数律描述了湍流边界层中间区域的通用速度分布。对数律是湍流边界层的核心特征，与层流边界层有本质区别。这种通用性使得对数律成为湍流边界层理论的基础。

实际应用例题
边界层理论
平板边界层计算
计算平板边界层的厚度和摩擦系数
解题过程：
已知：$L = 1 \text{ m}, U_\infty = 10 \text{ m/s}, \nu = 1.5 \times 10^{-5} \text{ m}^2/\text{s}$
雷诺数：$Re_L = \frac{U_\infty L}{\nu} = \frac{10 \times 1}{1.5 \times 10^{-5}} = 6.67 \times 10^5$
边界层厚度：$\delta = 5.0\sqrt{\frac{\nu x}{U_\infty}} = 5.0\sqrt{\frac{1.5 \times 10^{-5} \times 1}{10}} = 0.0061 \text{ m}$
位移厚度：$\delta^* = 1.72\sqrt{\frac{\nu x}{U_\infty}} = 0.0021 \text{ m}$
动量厚度：$\theta = 0.664\sqrt{\frac{\nu x}{U_\infty}} = 0.0008 \text{ m}$
摩擦系数：$C_f = \frac{0.664}{\sqrt{Re_x}} = \frac{0.664}{\sqrt{6.67 \times 10^5}} = 0.0026$
结果分析：
平板边界层厚度为6.1mm，位移厚度为2.1mm，动量厚度为0.8mm。摩擦系数为0.0026，表明边界层对平板阻力的贡献。这些参数是空气动力学设计的重要参考。
