第1章｜基础知识（PDF页 11-37）

本章页数：134 页
学习核心：海洋为什么能被看成旋转、分层、受边界强迫的薄层流体
原书逻辑：历史背景 → 海洋物理环境 → 大气动力作用 → 海洋热量收支 → 海洋物理参数

本章三句总纲

  • 本章建立海洋的几何尺度、边界条件和基础参数语言。
  • 热收支与温盐密结构是后续动力过程的初始背景。
  • 观测方法从一开始就是教材主线，而非附属内容。

公式 / 方法 / 现象锚点

  • 公式：海洋热量收支表达｜Q_net = Q_sw - Q_lw - Q_s - Q_l
  • 公式：声速经验关系｜C = C(T, S, Z)
  • 方法：海洋深度测量：回声测深给出高精度原位深度，卫星高度计通过海表形状反演海底重力异常与地形。
  • 方法：海洋内部探测：声学是几十米以下内部海洋信息的重要遥感手段。
  • 现象：四大洋与边缘海的面积和深度差异说明海盆几何背景并不统一。
  • 现象：海洋是薄层流体，水平尺度常比垂向尺度大三个量级。

课堂追问（可直接转练习）

  1. 教材强调海洋水平尺度远大于垂向尺度，因此海洋常被视为薄层流体。
  2. 决定海水动力状态的核心状态量常概括为温度、盐度和密度。
  3. 海洋热量收支中，与蒸发密切相关的冷却项是潜热通量。
  4. 教材用于高精度原位测深的主要方法是船载声学回声 / 回声测深。
  5. 海洋内部远距离传递信息的便利手段是声音 / 声学信号。
  6. 比原位温度更适合比较不同深度水体热状态的量是位温。

第2章｜物理海洋方程组（PDF页 38-63）

本章页数：72 页
学习核心：守恒律怎样被写成海洋中可近似、可诊断的方程组
原书逻辑：纳维－斯托克斯方程的建立 → 物理海洋方程的建立 → 物理海洋方程的处理方法

本章三句总纲

  • 第2章给出统一的方程语言。
  • 近似和尺度分析决定了后文每一种经典平衡。
  • 方程项的物理意义比方程外形更重要。

公式 / 方法 / 现象锚点

  • 公式：全导数 / 物质导数｜D/Dt = ∂/∂t + u∂/∂x + v∂/∂y + w∂/∂z
  • 公式：连续性方程｜∇·V = 0（不可压近似）
  • 方法：箱式模型：教材用陆架海区水量收支示例说明守恒律如何先从积分形式工作。
  • 方法：欧拉观点：把海洋分成固定控制体，研究流体穿越边界与场量变化。
  • 现象：海洋方程组中的非线性平流项，使解析求解在大多数情况下困难。
  • 现象：湍流参数化反映了海洋中大量次网格混合过程。

课堂追问（可直接转练习）

  1. 把离散分子流体看作连续场的基本前提叫连续介质假设。
  2. 欧拉观点下，变量随流体质点变化的导数常写成全 / 物质导数。
  3. 不可压海水最常用的连续性方程写成速度散度等于零。
  4. 将温盐结构与动力过程耦合起来的关系式是海水状态方程。
  5. 判断不同海洋过程该保留哪些方程项的方法叫尺度分析。
  6. 把速度场重写为更易诊断形式时，常用到流函数和涡度。

第3章｜潮汐（PDF页 64-82）

本章页数：75 页
学习核心：天文强迫如何经过海盆传播与局地地形改造成真实潮汐
原书逻辑：潮汐现象 → 潮汐基本理论 → 调和分析预报 → 潮波 → 潮汐观测 → 我国近海潮汐特征

本章三句总纲

  • 潮汐是最典型的天文强迫海洋过程。
  • 真实海洋潮汐必须通过潮波传播和边界效应理解。
  • 调和分析使潮汐从理论现象变成可预报对象。

公式 / 方法 / 现象锚点

  • 公式：引潮势｜潮汐强迫可由月、日引潮势展开表示
  • 公式：调和表示｜η(t) = Σ H_i cos(ω_i t + φ_i)
  • 方法：潮汐观测：验潮站获得长时间潮位序列，卫星高度计补充海盆尺度结构。
  • 方法：潮汐预报：通过分潮常数做调和分析与报潮。
  • 现象：全日潮、半日潮和混合潮体现海区响应差别。
  • 现象：潮汐不等现象体现轨道几何和地球自转的调制。

课堂追问（可直接转练习）

  1. 由日、月对海水的周期性引力作用所引起的海面起伏称为潮汐。
  2. 与海面起伏相伴随的海水水平运动称为潮流。
  3. 将复杂潮位分解成多个分潮叠加的方法叫潮汐调和分析。
  4. 理想条件下描述海洋对引潮势响应的理论称为平衡潮理论。
  5. 潮汐在海盆中的传播形式可称为潮波。
  6. 教材中对我国近海潮汐的应用讲解，最能体现海盆和地形对天文强迫的改造。

第4章｜地转流（PDF页 83-99）

本章页数：49 页
学习核心：压强梯度与科氏力如何在大尺度上形成近似平衡流
原书逻辑：静力平衡 → 地转方程组 → 卫星高度计计算 → 温盐资料计算

本章三句总纲

  • 地转流把海洋结构场与运动场连接起来。
  • 静力平衡和热成风是理解内部流速结构的关键。
  • 现代地转流诊断高度依赖高度计和温盐剖面观测。

公式 / 方法 / 现象锚点

  • 公式：静力平衡｜∂p/∂z = -ρg
  • 公式：地转平衡｜fv = (1/ρ)∂p/∂x，fu = -(1/ρ)∂p/∂y
  • 方法：卫星高度计：由海面高度异常或绝对动力地形反演表层地转流。
  • 方法：CTD / 温盐剖面：计算密度、比容异常和动力高度，进而估算相对地转流。
  • 现象：海表高度梯度对应表层地转流，卫星高度计因此能看见大尺度环流骨架。
  • 现象：水平密度梯度控制垂向剪切，是西边界流和中尺度涡旋的重要结构基础。

课堂追问（可直接转练习）

  1. 在大尺度海洋中，垂向压强梯度与重力平衡称为静力平衡。
  2. 大尺度水平流中，压强梯度力与科氏力平衡形成地转流。
  3. 把水平密度梯度与地转流垂向切变联系起来的关系叫热成风关系。
  4. 利用海表面几何信息反推表层地转流的重要观测手段是卫星高度计。
  5. 由温盐资料恢复内部地转流时，首先要把温盐转成密度结构。
  6. 不依赖静止参考面的绝对流估计教材中提到的方法是P矢量 / P 矢量方法。

第5章｜埃克曼流与惯性流（PDF页 100-118）

本章页数：62 页
学习核心：风应力、摩擦与自转怎样共同塑造表层偏转和惯性响应
原书逻辑：海表风应力 → 海表埃克曼层 → 埃克曼质量输运及其应用 → 海底埃克曼流 → 惯性流

本章三句总纲

  • 第5章把海表风应力转化成边界层动力学。
  • 真正重要的不是偏角口号，而是质量输运和垂向响应。
  • 埃克曼理论是风生海盆环流的直接前置。

公式 / 方法 / 现象锚点

  • 公式：海表风应力｜τ = ρ_a C_D |U| U
  • 公式：埃克曼输运｜M_E = τ / (ρ f)
  • 方法：风应力估算：由海面 10 m 风速和阻力系数转换为动量通量。
  • 方法：海表观测：漂流浮标、流速剖面和海表响应可检验埃克曼理论。
  • 现象：埃克曼螺旋说明表层以下流向会逐渐旋转并衰减。
  • 现象：沿岸上升流可带来低温高营养海水。

课堂追问（可直接转练习）

  1. 风把动量传给海洋上层的直接边界量叫海表风应力。
  2. 经典埃克曼层由风应力、科氏力和垂向摩擦共同控制。
  3. 比表层偏角更能代表风生边界层整体效应的量是埃克曼质量输运。
  4. 沿岸风使表层水离岸输运并诱发冷水上涌的现象称为沿岸上升流。
  5. 风应力旋度引起的垂向抽吸效应称为埃克曼抽吸。
  6. 外力撤去后海水在科氏力控制下做的自由振荡称为惯性流。

第6章｜风生大洋环流（PDF页 119-132）

本章页数：60 页
学习核心：风应力旋度如何经 Sverdrup 平衡和西边界强化塑造盆地环流
原书逻辑：斯韦德鲁普风生环流理论 → 西边界流理论 → 位势涡度守恒

本章三句总纲

  • 风生大洋环流是风场、旋转地球和边界共同作用的结果。
  • Sverdrup 理论给内部约束，西边界理论给闭合机制。
  • 位势涡度守恒提供更一般的旋转流体语言。

公式 / 方法 / 现象锚点

  • 公式：斯韦德鲁普平衡｜β V ≈ curl(τ)/(ρ_0)
  • 公式：位势涡度｜q = (ζ + f)/H
  • 方法：风场诊断：从风应力旋度场出发判断海盆内部 Sverdrup 输运符号与大小。
  • 方法：理论比较：线性底摩擦、水平黏性和惯性非线性分别对应不同西边界流模型。
  • 现象：副热带环流与副极地环流的方向与风带分布密切相关。
  • 现象：黑潮、湾流等西边界流表现为窄、强、深、快。

课堂追问（可直接转练习）

  1. 决定海盆内部大尺度输运的风场量，不是风速本身而是风应力旋度。
  2. 解释海盆内部风生输运的经典理论是斯韦德鲁普风生环流理论。
  3. 海盆环流闭合往往在西侧形成强而窄海流，这一现象称为西边界强化。
  4. 黑潮和湾流是现实海洋中典型的西边界流。
  5. 把相对涡度、行星涡度和流体柱厚度联系起来的守恒量是位势涡度。
  6. 位势涡度守恒最直观的物理图像是流体柱伸缩与自转变化相互补偿。

第7章｜深层环流理论（PDF页 133-143）

本章页数：49 页
学习核心：浮力强迫、混合与地形约束怎样共同组织深海翻转与水团传播
原书逻辑：深层环流结构 → 深层环流的重要性 → 深层环流理论 → 深层环流的观测

本章三句总纲

  • 深层环流是全球热量、碳和水团输送的慢闭环。
  • 翻转环流和深层西边界流提供结构与通道图像。
  • 多平衡态使其具有潜在的阈值和气候风险含义。

公式 / 方法 / 现象锚点

  • 公式：翻转环流表达（概念式）｜表层—深层之间存在经向质量与热量再分配
  • 公式：多平衡态的物理含义｜同一外部背景下，系统可能存在多个稳定状态
  • 方法：深层西边界流观测：利用长期系留和深海流速测量识别关键输送通道。
  • 方法：水文剖面与示踪物：通过深层水团性质追踪来源、年龄和交换路径。
  • 现象：深层环流控制全球热量长期再分配。
  • 现象：海洋固碳与深层水形成、下沉和再上翻密切相关。

课堂追问（可直接转练习）

  1. 深层环流对气候最直接的重要性之一是全球热量再分配。
  2. 教材强调深层环流除了热量作用外，还承担海水固碳作用。
  3. 连接表层与深层的大尺度经向交换常概括为翻转环流。
  4. 深层输送在盆地内部常通过狭窄的深层西边界流完成。
  5. 如果同样背景下系统可处于多个稳定状态，这一性质称为多平衡态。
  6. 深层环流之所以被称为气候系统的慢变量，是因为其调整时间尺度通常较长。

第8章｜波浪理论（PDF页 144-167）

本章页数：124 页
学习核心：恢复力怎样决定波动传播、频散、能量输运与浅水变形
原书逻辑：波动现象 → 线性波理论 → 非线性波理论 → 浅水变形/折射/绕射/反射/破碎 → 随机波浪理论 → 观测 → 极端灾害性波动

本章三句总纲

  • 波浪理论是从控制方程到工程应用最完整的一章。
  • 相速、群速、波能流和波谱构成理解波场的核心语言。
  • 近岸变形和灾害波动让理论直接走向现实场景。

公式 / 方法 / 现象锚点

  • 公式：线性表面重力波色散关系｜ω² = gk tanh(kh)
  • 公式：波能｜E ≈ (1/8)ρgH²
  • 方法：规则波分析：从控制方程、边界条件和色散关系求解基础波场。
  • 方法：波谱诊断：用随机波浪理论描述真实海况中的频率能量分布。
  • 现象：相速和群速不一致，说明海浪传播的不只是波峰位置。
  • 现象：浅水折射、绕射和破碎塑造近岸波场。

课堂追问（可直接转练习）

  1. 在线性波理论中，描述相位传播快慢的速度称为相速。
  2. 在线性波理论中，更能代表能量传播的速度称为群速。
  3. 把实际海浪看作统计波场时，最常用的描述工具是波浪谱。
  4. 波浪进入浅水区后因水深变化而改变传播方向的现象称为折射。
  5. 由强天气系统引起的海面显著增水和灾害性波动称为风暴潮。
  6. 由海底地震等激发、以长波形式传播的典型灾害性海洋波动是海啸。

第9章｜海洋中的大尺度波动（PDF页 168-177）

本章页数：14 页
学习核心：β 效应与旋转约束如何产生 Kelvin 波、Rossby 波等大尺度调整过程
原书逻辑：线性波动理论 → 庞加莱波 → 开尔文波 → 罗斯贝波 → 地形罗斯贝波

本章三句总纲

  • 大尺度波动是海洋信息传播和盆地调整的重要方式。
  • 不同波型的差异来自恢复机制、边界条件和时间尺度不同。
  • Kelvin 波与 Rossby 波是连接 ENSO 和海盆调整的关键对象。

公式 / 方法 / 现象锚点

  • 公式：Kelvin 波核心特征｜沿边界传播，横向速度近零，振幅离边界指数衰减
  • 公式：Rossby 波色散（概念式）｜频率受 β 与波数控制，能量主要承担慢调整
  • 方法：理论诊断：先用线性波动框架区分不同恢复机制。
  • 方法：物理比较：边界约束、β 效应和地形梯度分别对应不同波型。
  • 现象：庞加莱波体现旋转背景下的惯性重力波特征。
  • 现象：Kelvin 波承担沿岸和赤道快速信息传播。

课堂追问（可直接转练习）

  1. 兼具旋转约束和重力恢复的大尺度基础波型是庞加莱波。
  2. 沿岸或赤道快速传播、受边界束缚的长波称为开尔文 / Kelvin波。
  3. 以 β 效应为核心恢复机制并承担慢调整的波型是罗斯贝 / Rossby波。
  4. 若恢复机制来自海底坡度等地形效应，可形成地形罗斯贝波。
  5. Kelvin 波之所以特殊，是因为传播受到边界条件强约束。
  6. Rossby 波虽然传播慢，却非常适合承担海盆尺度的长期调整。

第10章｜海洋内波（PDF页 178-194）

本章页数：76 页
学习核心：层化海洋中的浮力恢复怎样形成内部波、内潮和内孤立波
原书逻辑：内波现象 → 基本控制方程 → 频散关系 → 动力学特征 → 反射与折射 → 观测 → 内潮和内孤立波

本章三句总纲

  • 内波是层化海洋的核心波动形式。
  • N²、频散关系和本征模态构成理论主线。
  • 内潮与内孤立波把跨章节知识压缩成最典型的可观测对象。

公式 / 方法 / 现象锚点

  • 公式：稳定度与浮力频率｜N² = -(g/ρ) dρ/dz
  • 公式：内波频散关系｜ω 与 k、m、N、f 之间存在层化控制的色散关系
  • 方法：层化诊断：先用温盐剖面计算密度和浮力频率。
  • 方法：理论求解：利用本征方程与频散关系判断可传播模态。
  • 现象：死水现象是内波最经典的历史案例。
  • 现象：稳定层化、能量来源和激发源共同决定内波生成。

课堂追问（可直接转练习）

  1. 内波存在的前提是海洋内部具有稳定的层化 / 密度分层。
  2. 衡量层结强度并限定内波频率范围的重要量是浮力频率。
  3. 把复杂内波场拆分为不同垂向结构时使用的是本征模态。
  4. 由潮汐流经起伏地形激发的典型内波称为内潮。
  5. 具有强非线性和大振幅特征的典型内波形式是内孤立波。
  6. 内波之所以重要，是因为它连接大尺度能量输入与小尺度混合增强。
