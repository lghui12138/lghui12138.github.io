第1章｜基础知识（PDF页 11-37）

本章页数：134 页
学习核心：海洋为什么能被看成旋转、分层、受边界强迫的薄层流体
原书逻辑：历史背景 → 海洋物理环境 → 大气动力作用 → 海洋热量收支 → 海洋物理参数

本章三句总纲

  • 本章建立海洋的几何尺度、边界条件和基础参数语言。
  • 热收支与温盐密结构是后续动力过程的初始背景。
  • 观测方法从一开始就是教材主线，而非附属内容。

公式 / 方法 / 现象锚点

  • 公式：海洋热量收支表达｜Q_net = Q_sw - Q_lw - Q_s - Q_l
  • 公式：声速经验关系｜C = C(T, S, Z)
  • 方法：海洋深度测量：回声测深给出高精度原位深度，卫星高度计通过海表形状反演海底重力异常与地形。
  • 方法：海洋内部探测：声学是几十米以下内部海洋信息的重要遥感手段。
  • 现象：四大洋与边缘海的面积和深度差异说明海盆几何背景并不统一。
  • 现象：海洋是薄层流体，水平尺度常比垂向尺度大三个量级。

课堂追问（可直接转练习）

  1. 教材强调海洋水平尺度远大于垂向尺度，因此海洋常被视为薄层流体。
  2. 决定海水动力状态的核心状态量常概括为温度、盐度和密度。
  3. 海洋热量收支中，与蒸发密切相关的冷却项是潜热通量。
  4. 教材用于高精度原位测深的主要方法是船载声学回声 / 回声测深。
  5. 海洋内部远距离传递信息的便利手段是声音 / 声学信号。
  6. 比原位温度更适合比较不同深度水体热状态的量是位温。

第2章｜物理海洋方程组（PDF页 38-63）

本章页数：72 页
学习核心：守恒律怎样被写成海洋中可近似、可诊断的方程组
原书逻辑：纳维－斯托克斯方程的建立 → 物理海洋方程的建立 → 物理海洋方程的处理方法

本章三句总纲

  • 第2章给出统一的方程语言。
  • 近似和尺度分析决定了后文每一种经典平衡。
  • 方程项的物理意义比方程外形更重要。

公式 / 方法 / 现象锚点

  • 公式：全导数 / 物质导数｜D/Dt = ∂/∂t + u∂/∂x + v∂/∂y + w∂/∂z
  • 公式：连续性方程｜∇·V = 0（不可压近似）
  • 方法：箱式模型：教材用陆架海区水量收支示例说明守恒律如何先从积分形式工作。
  • 方法：欧拉观点：把海洋分成固定控制体，研究流体穿越边界与场量变化。
  • 现象：海洋方程组中的非线性平流项，使解析求解在大多数情况下困难。
  • 现象：湍流参数化反映了海洋中大量次网格混合过程。

课堂追问（可直接转练习）

  1. 把离散分子流体看作连续场的基本前提叫连续介质假设。
  2. 欧拉观点下，变量随流体质点变化的导数常写成全 / 物质导数。
  3. 不可压海水最常用的连续性方程写成速度散度等于零。
  4. 将温盐结构与动力过程耦合起来的关系式是海水状态方程。
  5. 判断不同海洋过程该保留哪些方程项的方法叫尺度分析。
  6. 把速度场重写为更易诊断形式时，常用到流函数和涡度。

第3章｜潮汐（PDF页 64-82）

本章页数：75 页
学习核心：天文强迫如何经过海盆传播与局地地形改造成真实潮汐
原书逻辑：潮汐现象 → 潮汐基本理论 → 调和分析预报 → 潮波 → 潮汐观测 → 我国近海潮汐特征

本章三句总纲

  • 潮汐是最典型的天文强迫海洋过程。
  • 真实海洋潮汐必须通过潮波传播和边界效应理解。
  • 调和分析使潮汐从理论现象变成可预报对象。

公式 / 方法 / 现象锚点

  • 公式：引潮势｜潮汐强迫可由月、日引潮势展开表示
  • 公式：调和表示｜η(t) = Σ H_i cos(ω_i t + φ_i)
  • 方法：潮汐观测：验潮站获得长时间潮位序列，卫星高度计补充海盆尺度结构。
  • 方法：潮汐预报：通过分潮常数做调和分析与报潮。
  • 现象：全日潮、半日潮和混合潮体现海区响应差别。
  • 现象：潮汐不等现象体现轨道几何和地球自转的调制。

课堂追问（可直接转练习）

  1. 由日、月对海水的周期性引力作用所引起的海面起伏称为潮汐。
  2. 与海面起伏相伴随的海水水平运动称为潮流。
  3. 将复杂潮位分解成多个分潮叠加的方法叫潮汐调和分析。
  4. 理想条件下描述海洋对引潮势响应的理论称为平衡潮理论。
  5. 潮汐在海盆中的传播形式可称为潮波。
  6. 教材中对我国近海潮汐的应用讲解，最能体现海盆和地形对天文强迫的改造。

第4章｜地转流（PDF页 83-99）

本章页数：49 页
学习核心：压强梯度与科氏力如何在大尺度上形成近似平衡流
原书逻辑：静力平衡 → 地转方程组 → 卫星高度计计算 → 温盐资料计算

本章三句总纲

  • 地转流把海洋结构场与运动场连接起来。
  • 静力平衡和热成风是理解内部流速结构的关键。
  • 现代地转流诊断高度依赖高度计和温盐剖面观测。

公式 / 方法 / 现象锚点

  • 公式：静力平衡｜∂p/∂z = -ρg
  • 公式：地转平衡｜fv = (1/ρ)∂p/∂x，fu = -(1/ρ)∂p/∂y
  • 方法：卫星高度计：由海面高度异常或绝对动力地形反演表层地转流。
  • 方法：CTD / 温盐剖面：计算密度、比容异常和动力高度，进而估算相对地转流。
  • 现象：海表高度梯度对应表层地转流，卫星高度计因此能看见大尺度环流骨架。
  • 现象：水平密度梯度控制垂向剪切，是西边界流和中尺度涡旋的重要结构基础。

课堂追问（可直接转练习）

  1. 在大尺度海洋中，垂向压强梯度与重力平衡称为静力平衡。
  2. 大尺度水平流中，压强梯度力与科氏力平衡形成地转流。
  3. 把水平密度梯度与地转流垂向切变联系起来的关系叫热成风关系。
  4. 利用海表面几何信息反推表层地转流的重要观测手段是卫星高度计。
  5. 由温盐资料恢复内部地转流时，首先要把温盐转成密度结构。
  6. 不依赖静止参考面的绝对流估计教材中提到的方法是P矢量 / P 矢量方法。

第5章｜埃克曼流与惯性流（PDF页 100-118）

本章页数：62 页
学习核心：风应力、摩擦与自转怎样共同塑造表层偏转和惯性响应
原书逻辑：海表风应力 → 海表埃克曼层 → 埃克曼质量输运及其应用 → 海底埃克曼流 → 惯性流

本章三句总纲

  • 第5章真正要记的不是“北右南左”口号，而是风应力怎样经垂向摩擦转成整层净输运。
  • 一旦分清“表层流向”与“整层输运”不是一回事，上升流、下沉流和埃克曼抽吸就能顺成一条链。
  • 惯性流说明即使外力撤去，旋转海洋也会以当地惯性频率继续响应。

公式 / 方法 / 现象锚点

  • 公式：海表风应力｜τ = ρa CD |U10| U10；10 m 风速越大，向海洋输入的动量通量通常越强。
  • 公式：埃克曼质量输运｜ME = (k × τ) / (ρ0 f)；北半球净输运在风向右侧 90°，南半球相反，但这不是最表层流向。
  • 方法：边界层判读：先看风向与岸线夹角，再判断表层净输运是离岸还是向岸，最后再推上升流或下沉流。
  • 方法：观测核对：漂流浮标、ADCP、SST 与 Chl-a 常联合验证“风起—输运变—近岸冷水出露”这条链。
  • 现象：埃克曼螺旋反映速度随深度旋转并衰减，但真实海洋常因层化、非定常和湍混合而偏离理想解。
  • 现象：沿岸上升流并不是“有风就有”，还受岸线朝向、层化强弱、陆架宽度和背景流共同控制。

课堂追问（可直接转练习）

  1. 经典埃克曼层中，控制净输运方向的不是表层瞬时流向，而是整层积分后的埃克曼质量输运。
  2. 北半球沿岸若风使表层净输运离岸，近岸下层冷水常上涌形成上升流。
  3. 若风向和岸线配置使表层水向岸堆积，常诱发下沉流而不是上升流。
  4. 埃克曼抽吸的直接动力来源不是风速本身，而是风应力旋度。
  5. 经典惯性流在忽略摩擦时，其振荡频率等于当地惯性频率 / 科氏参数。
  6. 判断一道沿岸上升流题时，最稳妥顺序是风向—净输运—近岸质量收支—温盐营养响应。

第6章｜风生大洋环流（PDF页 119-132）

本章页数：60 页
学习核心：风应力旋度如何经 Sverdrup 平衡和西边界强化塑造盆地环流
原书逻辑：斯韦德鲁普风生环流理论 → 西边界流理论 → 位势涡度守恒

本章三句总纲

  • 风生大洋环流真正的闭环逻辑是：风应力旋度先规定海盆内部 Sverdrup 输运，再由西边界流把质量和涡度账补平。
  • 西边界强化不能只背“黑潮很强”，要能说出 β 效应、盆地边界和摩擦闭合为什么缺一不可。
  • 位势涡度守恒把内区输运、边界流调整和地形效应串成同一种旋转流体语言。

公式 / 方法 / 现象锚点

  • 公式：Sverdrup 平衡｜β My = curlz(τ) / ρ0；它约束的是海盆内部经向输运，不是直接给出某一点的流速细节。
  • 公式：位势涡度｜q = (ζ + f) / H；流体柱南北移动或厚度变化时，必须通过相对涡度变化来补偿。
  • 方法：海盆诊断：先看风应力旋度符号，再判断副热带 / 副极地环流方向，最后补边界流闭合。
  • 方法：理论比较：Stommel 模型强调底摩擦闭合，Munk 模型强调水平黏性控制西边界层宽度。
  • 现象：黑潮、湾流之所以“窄而强”，不是风只吹西边，而是内区积累的行星涡度变化要在西边界薄层内被迅速调整。
  • 现象：海盆内部大多近 Sverdrup 平衡，但西边界流、分离流和再循环区往往最不“教科书”。

课堂追问（可直接转练习）

  1. Sverdrup 理论直接约束的是海盆内部经向输运，而不是把整盆地流场一次性全部解完。
  2. 副热带环流闭合必须引入西边界强化，否则内区输运无法与边界条件同时满足。
  3. 黑潮和湾流属于西边界流，其“强、窄、深”本质上是 β 效应与摩擦闭合共同结果。
  4. 位势涡度守恒最常见图像是流体柱南北移动或伸缩时，相对涡度相应调整。
  5. 只看风速大小而不看风应力旋度，往往无法判断海盆内部 Sverdrup 输运方向。
  6. 盆地环流题若只背“北顺南逆”，但讲不出西边界强化原因，通常拿不到高分。

第7章｜深层环流理论（PDF页 133-143）

本章页数：49 页
学习核心：浮力强迫、混合与地形约束怎样共同组织深海翻转与水团传播
原书逻辑：深层环流结构 → 深层环流的重要性 → 深层环流理论 → 深层环流的观测

本章三句总纲

  • 深层环流不是“深海里慢慢流”，而是由高纬深水形成、盆地尺度传播和低纬 / 南大洋上翻共同闭合的全球翻转系统。
  • 深层西边界流是北大西洋深层水等深水团向低纬输出的主通道，地形约束和位势涡度收支决定其常呈贴边、窄而集中的结构。
  • 多平衡态意味着淡水强迫、表层增密或混合效率的改变，不只会让翻转环流“强一点或弱一点”，还可能推动系统跨过阈值而重组。

公式 / 方法 / 现象锚点

  • 公式：翻转流函数｜Ψ(y,z) = ∫-Hz ∫ v(x,y,z') dxdz'；常用其正负号、极值和深度位置描述经向翻转环流强弱与结构。
  • 公式：位势涡度约束（概念式）｜q = (f + ζ) / h；深层水沿地形传播时常受位势涡度近守恒约束，因此更易组织成深层西边界流和地形跟随流。
  • 方法：深层西边界流观测：利用 RAPID / OSNAP 一类断面阵列、深海系留、ADCP 与底层流速仪，持续监测翻转强度和关键通道输送。
  • 方法：水文剖面与示踪物：把温盐、溶解氧、CFC、放射性碳等信息结合起来，才能同时判断深层水来源、年龄、混合与传播路径。
  • 现象：北大西洋深层水和南极底层水并不是均匀铺满全球海盆，而是沿深层西边界和地形通道优先生长、传播与交换。
  • 现象：深层环流影响的不只是热量再分配，还直接关系海洋储碳、深海氧供给以及气候系统的长期记忆。

课堂追问（可直接转练习）

  1. 北大西洋和南极边缘海之所以成为深水形成关键区，核心不是“纬度高”本身，而是强表层浮力损失能使表层水持续增密并下沉。
  2. 翻转环流的“闭环”不能只说高纬下沉，还必须补出深层传播、上翻以及表层回流这几段链条。
  3. 深层输送常沿深层西边界流集中，不是因为海盆内部完全不流，而是因为位势涡度约束、地形引导和摩擦闭合共同让输送更容易贴边组织。
  4. 判断深层环流是否变化，不能只盯某一处温度，还要结合断面输送、密度结构和示踪物年龄等多类证据。
  5. 所谓多平衡态，是指同样外部背景下翻转环流可能存在多个稳定解，因此淡水扰动可能触发突变式调整。
  6. 深层环流被称为气候系统慢变量，正是因为它一旦改变，恢复和重建往往需要远长于天气和季节尺度的时间。

第8章｜波浪理论（PDF页 144-167）

本章页数：124 页
学习核心：恢复力怎样决定波动传播、频散、能量输运与浅水变形
原书逻辑：波动现象 → 线性波理论 → 非线性波理论 → 浅水变形/折射/绕射/反射/破碎 → 随机波浪理论 → 观测 → 极端灾害性波动

本章三句总纲

  • 线性波理论真正给出的不是“背一个公式”，而是把自由面问题压缩成色散、能量传播和边界条件三条主线。
  • 判断深水波、浅水波和近岸变形时，核心不是主观感觉“水深不深”，而是先看 kh 控制下的传播机制是否改变。
  • 会答题的人不仅能写出相速、群速和波能，还能说清波浪为什么会在近岸浅化、折射、增高并最终破碎。

公式 / 方法 / 现象锚点

  • 公式：线性表面重力波色散关系｜ω2 = gk tanh(kh)；它把波速是否受水深控制这件事一次讲清。
  • 公式：相速与群速｜c = ω/k，cg = dω/dk；深水重力波中常有 cg = c/2，浅水长波则近似 c ≈ cg ≈ √(gh)。
  • 方法：波型判读：先用 kh 区分深水 / 过渡水深 / 浅水，再决定该用频散、射线还是浅水近似去解释现象。
  • 方法：随机海况诊断：真实海浪通常不能只看单一规则波，要结合波谱、显著波高和主频带去判断能量分布。
  • 现象：波峰移动快慢不等于能量传播快慢；很多“看起来跑得很快”的波场，真正输送能量的速度由群速决定。
  • 现象：海啸虽然在深海中波高不大，却因波长极长而依旧属于浅水长波；风暴潮则主要是风应力和气压场抬升海面，不应和海啸混成一类。

课堂追问（可直接转练习）

  1. 若 kh ≫ 1，通常进入深水波近似，此时波速主要受波长控制而不是直接受水深控制。
  2. 深水重力波里常见的关键结论不是“相速最大”，而是群速约为相速的一半，因此能量传播慢于波峰前进。
  3. 若 kh ≪ 1，浅水长波波速近似 √(gh)，说明水深比波长更直接决定传播快慢。
  4. 波浪进入浅水区后若海底地形使波射线收敛，局地波高往往升高，这正是折射和浅化共同作用的结果。
  5. 海啸在深海中仍可按浅水长波近似处理，关键原因是其波长远大于海洋平均水深。
  6. 风暴潮的主驱动是风应力与气压场抬升海面，而海啸的主激发多与海底地震、滑坡等突发位移有关。

第9章｜海洋中的大尺度波动（PDF页 168-177）

本章页数：14 页
学习核心：β 效应与旋转约束如何产生 Kelvin 波、Rossby 波等大尺度调整过程
原书逻辑：线性波动理论 → 庞加莱波 → 开尔文波 → 罗斯贝波 → 地形罗斯贝波

本章三句总纲

  • 这一章真正要抓住的是：同样是“海洋波动”，庞加莱波、Kelvin 波和 Rossby 波的恢复机制、边界约束和时间尺度完全不同。
  • Kelvin 波负责沿岸或赤道的快速调整，Rossby 波负责盆地尺度的慢调整与记忆；二者配合才构成很多海盆响应的完整闭环。
  • 只背“Kelvin 快、Rossby 慢”远远不够，还要知道束缚宽度、传播方向和 β 效应为何会把海盆调整做成明显非对称。

公式 / 方法 / 现象锚点

  • 公式：Kelvin 波束缚尺度｜η, u ∝ exp(-y/Rd)，其中 Rd = c/f；说明其振幅离海岸或赤道会在一个变形半径量级内迅速衰减。
  • 公式：长 Rossby 波相速（概念式）｜cx ≈ -βRd2；负号提示其中低纬海盆里最常见的是向西相位传播。
  • 方法：波型辨识：先问恢复机制来自重力、旋转还是 β 效应，再问是否受海岸 / 赤道束缚，最后才谈传播方向和时间尺度。
  • 方法：ENSO 诊断：东风异常、热跃层倾斜、赤道 Kelvin 波东传和 Rossby 波西传—边界反射，是最常见的答题主线。
  • 现象：赤道 Kelvin 波虽然传播快，但并不是“整条赤道都一样强”，其振幅仍受变形半径控制并集中在赤道附近。
  • 现象：Rossby 波承担的不是局地短时振荡，而是把行星涡度约束转成海盆尺度的慢调整和长期记忆。

课堂追问（可直接转练习）

  1. 兼具旋转约束和重力恢复的大尺度基础波型是庞加莱波，它更像 Kelvin 波与 Rossby 波讨论前的总入口。
  2. 沿岸或赤道快速传播、振幅离边界指数衰减的束缚长波称为开尔文 / Kelvin 波。
  3. 若题目强调 β 效应、慢调整和海盆记忆，优先想到的应是罗斯贝 / Rossby 波而不是 Kelvin 波。
  4. Kelvin 波的横向束缚宽度通常由变形半径控制，因此它不是“贴着边界无限窄地跑”。
  5. 长 Rossby 波在多数中低纬海盆里常表现为向西相位传播，这一点是和 Kelvin 波最稳的区分点之一。
  6. ENSO 相关题若只说海温异常却不补赤道 Kelvin 波和 Rossby 波的传播—反射链，通常答案不算完整。

第10章｜海洋内波（PDF页 178-194）

本章页数：76 页
学习核心：层化海洋中的浮力恢复怎样形成内部波、内潮和内孤立波
原书逻辑：内波现象 → 基本控制方程 → 频散关系 → 动力学特征 → 反射与折射 → 观测 → 内潮和内孤立波

本章三句总纲

  • 内波不是海面小波纹的“水下版”，而是层化海洋在浮力恢复作用下的内部振荡。
  • 先抓住 N2、频率范围和模态结构，再谈内潮与内孤立波就不容易散。
  • 真正会答这一章的人，能把“层化—激发—传播—破碎混合”连成一条完整物理链。

公式 / 方法 / 现象锚点

  • 公式：稳定度与浮力频率｜N2 = -(g / ρ0) ∂ρ / ∂z；稳定层化时 N2 > 0。
  • 公式：自由内波频率范围｜在稳定层化旋转海洋中，通常满足 f < ω < N；低于 f 难以传播，高于 N 会被层化抑制。
  • 方法：层化诊断：先用 CTD / 温盐剖面求密度和浮力频率，再判断跃层位置与内波活跃层。
  • 方法：模态判读：把观测到的流速、位移和温度信号投影到本征模态，才能分清一阶、二阶等垂向结构。
  • 现象：正压潮流扫过海脊、陆坡和海岭时，最容易把能量转换成内潮，再在传播中非线性陡化为内孤立波。
  • 现象：内波的重要性不只在“会传播”，更在于其破碎后能把大尺度能量转成小尺度混合。

课堂追问（可直接转练习）

  1. 内波区别于表面波的关键，不是“在水里传播”，而是恢复力主要来自浮力和层化。
  2. 只有稳定层化存在时，自由内波才有清楚的传播频带和模态结构。
  3. 浮力频率越大，通常说明层结越强，内波允许的最高频率也越高。
  4. 当正压潮流过海脊或陆坡时，最容易激发的典型内部波动是内潮。
  5. 内孤立波常由内潮在地形和非线性作用下进一步陡化形成。
  6. 内波之所以重要，不仅因为它能传播，还因为它能在破碎后显著增强垂向混合。

第11章｜若干前沿专题（PDF页 195-221）

本章页数：97 页
学习核心：从经典理论走向真实海洋中的多尺度耦合、观测与预报
原书逻辑：赤道过程 → 极地过程 → 海洋中尺度涡旋 → 海洋次中尺度过程

本章三句总纲

  • 前沿专题不是额外背诵材料，而是把前 10 章的平衡、波动、环流和层化语言真正搬进真实海洋问题。
  • 赤道、极地、中尺度和次中尺度看似分散，实则分别对应海气耦合、浮力强迫、近地转涡旋和强非地转锋生四类典型研究语境。
  • 这一章最重要的能力是用尺度分析选对物理图像：什么时候近地转够用，什么时候必须把混合层、锋生和垂向速度重新拉回主线。

公式 / 方法 / 现象锚点

  • 公式：Rossby 数｜Ro = U / (fL)；中尺度通常 Ro ≪ 1、近地转近似较好，次中尺度常上升到 Ro = O(1)，非地转过程显著增强。
  • 公式：变形半径｜Rd ≈ NH / f；它决定锋、涡和波动从“受层化约束”到“能跨多大水平尺度”这一关键门槛。
  • 方法：综合观测：TAO / TRITON、Argo、卫星高度计、海色、滑翔机和系留观测要联合用，单一资料通常只能看见问题的一部分。
  • 方法：涡旋与锋识别：海表高度、相对涡度、海表温盐、混合层深度和拉格朗日漂流资料结合，才更容易分清中尺度涡旋与次中尺度锋生事件。
  • 现象：赤道 Kelvin 波东传、Rossby 波西传以及海气反馈共同决定 ENSO 的传播、成熟和衰减节奏。
  • 现象：海冰生长排盐、融化淡化，中尺度涡旋重排热盐碳，次中尺度锋生增强垂向交换，这些过程一起把“经典理论”推进到真实气候系统。

课堂追问（可直接转练习）

  1. 海气耦合最典型、最著名的赤道年际振荡现象缩写为 ENSO，而不是单独某一种海温或风场异常的名字。
  2. 教材中用于衡量南方涛动强弱的重要量是南方涛动指数，它和 Niño 指数一起才能把海气两侧都交代完整。
  3. 极地过程与海冰变化、深水形成及全球翻转环流背景密切相关，因此绝不是“只和极地局地天气有关”。
  4. 海洋中尺度涡旋研究中，自动识别很重要，但最终仍需结合生命周期、背景流和水团结构做物理判读。
  5. 相较中尺度，次中尺度过程通常具有更强的非地转成分、锋生作用和垂向交换。
  6. 前沿专题章节的根本作用，是训练你把基础理论迁移到真实海洋与现代科研问题，而不是额外背几段时事材料。

全书闭环

复习用途：把 11 章串成一条答题主线
学习核心：把“状态量—守恒律—经典平衡—环流与波动—真实海洋前沿”连成可复述、可做题、可迁移的整体框架
原书逻辑：先认识海洋的温盐密结构与观测语言 → 再写成方程和近似 → 再进入潮汐、地转、埃克曼、风生环流等经典动力过程 → 最后走到大尺度波动、内波和多尺度前沿问题

本章三句总纲

  • 物理海洋学不是把章节背散，而是把“驱动—响应—观测—诊断”四件事反复套用到不同过程。
  • 温盐密决定层化背景，方程和近似决定能不能解，平衡和边界层决定流怎么组织起来。
  • 真正会答题的标志，是能把某个现象同时讲清驱动、主控力、典型观测证据和可能后果。

公式 / 方法 / 现象锚点

  • 公式：状态方程主线｜ρ = ρ(T, S, p)，它把热盐结构与动力过程真正连起来。
  • 公式：三类经典平衡｜静力平衡管垂向，地转平衡管大尺度水平，埃克曼理论管风生边界层与垂向抽吸。
  • 方法：答题顺序｜先交代尺度与层化，再判断主导平衡，最后补观测手段和典型现象。
  • 方法：资料判读｜海表高度看表层地转流，CTD / Argo 看温盐密剖面，系留 / ADCP 看流速结构，卫星看盆地到次中尺度表征。
  • 现象：上升流、锋面、涡旋、潮波、内波看似分散，实质上都在回答海洋怎样在旋转、分层和边界约束下调整自身。
  • 现象：从 ENSO 到深层翻转环流，时间尺度越长，越需要把海气耦合、混合和地形约束一起考虑。

课堂追问（可直接转练习）

  1. 若题目先给温度和盐度结构，第一反应应是先判断密度分层和稳定层化，而不是直接背流型名称。
  2. 若题目问大尺度海流主控机制，通常应优先判断是否满足静力—地转近似，再讨论热成风和边界效应。
  3. 若题目问沿岸上升流，答题骨架至少要包含沿岸风、埃克曼离岸输运、下层补偿上涌和冷水高营养信号。
  4. 若题目问盆地尺度环流闭合，核心关键词通常是风应力旋度、Sverdrup 输运和西边界强化。
  5. 若题目问大尺度异常如何传播，首先想到的波动对象应包括 Kelvin 波和 Rossby 波，而不是把所有波都混成表面重力波。
  6. 若题目要求总结全书，可用“温盐密背景—守恒方程—经典平衡—环流波动—观测与前沿”五段式收束。
