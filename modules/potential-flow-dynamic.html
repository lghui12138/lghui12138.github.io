势流基本概念

势流理论是理想流体力学的基础，研究无旋、不可压缩流体的流动。势流具有速度势函数和流函数，可以通过叠加基本流动来构造复杂流动。

速度势函数
势流理论
$$\vec{v} = \nabla \phi$$
速度势函数与速度场的关系
推导过程：
对于无旋流动：$\nabla \times \vec{v} = 0$
引入速度势函数：$\vec{v} = \nabla \phi$
分量形式：$u = \frac{\partial \phi}{\partial x}, v = \frac{\partial \phi}{\partial y}$
不可压缩条件：$\nabla \cdot \vec{v} = 0$
代入得到：$\nabla^2 \phi = 0$（拉普拉斯方程）
物理意义：
速度势函数是标量函数，其梯度等于速度场。势函数等值线垂直于流线，描述了流体的运动方向。
势流理论 速度势 无旋流动
流函数
势流理论
$$u = \frac{\partial \psi}{\partial y}, \quad v = -\frac{\partial \psi}{\partial x}$$
流函数与速度分量的关系
推导过程：
不可压缩条件：$\frac{\partial u}{\partial x} + \frac{\partial v}{\partial y} = 0$
引入流函数：$u = \frac{\partial \psi}{\partial y}, v = -\frac{\partial \psi}{\partial x}$
自动满足不可压缩条件
对于无旋流动：$\frac{\partial v}{\partial x} - \frac{\partial u}{\partial y} = 0$
代入得到：$\nabla^2 \psi = 0$（拉普拉斯方程）
物理意义：
流函数是标量函数，其等值线表示流线。流函数值相等的点构成流线，描述了流体的运动轨迹。
势流理论 流函数 流线
复势函数
势流理论
$$W(z) = \phi + i\psi$$
复势函数，结合速度势和流函数
推导过程：
复变量：$z = x + iy$
复势函数：$W(z) = \phi(x,y) + i\psi(x,y)$
复速度：$\frac{dW}{dz} = u - iv$
柯西-黎曼条件：$\frac{\partial \phi}{\partial x} = \frac{\partial \psi}{\partial y}, \frac{\partial \phi}{\partial y} = -\frac{\partial \psi}{\partial x}$
复势函数是解析函数，满足拉普拉斯方程
物理意义：
复势函数是势流理论的重要工具，将速度势和流函数统一在复变函数理论框架下。复速度的模表示速度大小，辐角表示速度方向。
势流理论 复势函数 复变函数
均匀流
势流理论
$$W(z) = U_\infty z$$
均匀流的复势函数
推导过程：
均匀流速度：$u = U_\infty, v = 0$
速度势：$\phi = U_\infty x$
流函数：$\psi = U_\infty y$
复势函数：$W(z) = U_\infty x + iU_\infty y = U_\infty z$
复速度：$\frac{dW}{dz} = U_\infty$
物理意义：
均匀流是最基本的势流，速度场为常数。均匀流可以作为其他复杂流动的基础，通过叠加得到更复杂的流动。
势流理论 均匀流 基本流动
点源和点汇
势流理论
$$W(z) = \frac{m}{2\pi}\ln(z - z_0)$$
点源（m>0）或点汇（m<0）的复势函数
推导过程：
极坐标：$z - z_0 = re^{i\theta}$
复势函数：$W(z) = \frac{m}{2\pi}\ln(re^{i\theta}) = \frac{m}{2\pi}(\ln r + i\theta)$
速度势：$\phi = \frac{m}{2\pi}\ln r$
流函数：$\psi = \frac{m}{2\pi}\theta$
径向速度：$v_r = \frac{m}{2\pi r}$
切向速度：$v_\theta = 0$
物理意义：
点源和点汇是势流理论的基本流动。点源表示流体从一点向外流出，点汇表示流体向一点汇聚。径向速度随距离的倒数衰减，流线为从源点发出的射线。
势流理论 点源点汇 基本流动
点涡
势流理论
$$W(z) = \frac{\Gamma}{2\pi i}\ln(z - z_0)$$
点涡的复势函数，Γ为环量
推导过程：
极坐标：$z - z_0 = re^{i\theta}$
复势函数：$W(z) = \frac{\Gamma}{2\pi i}\ln(re^{i\theta}) = \frac{\Gamma}{2\pi i}(\ln r + i\theta)$
速度势：$\phi = -\frac{\Gamma}{2\pi}\theta$
流函数：$\psi = \frac{\Gamma}{2\pi}\ln r$
径向速度：$v_r = 0$
切向速度：$v_\theta = \frac{\Gamma}{2\pi r}$
物理意义：
点涡表示围绕一点的旋转流动，环量Γ表示旋转强度。切向速度随距离的倒数衰减，径向速度为零。流线为同心圆，等势线为从涡心发出的射线。
势流理论 点涡 环量
偶极子
势流理论
$$W(z) = \frac{\mu}{2\pi}\frac{1}{z - z_0}$$
偶极子的复势函数，μ为偶极矩
推导过程：
偶极子定义：点源和点汇的极限
$W(z) = \frac{m}{2\pi}[\ln(z - z_0 + \frac{\delta}{2}) - \ln(z - z_0 - \frac{\delta}{2})]$
极限过程：$\delta \to 0, m \to \infty, m\delta = \mu$
$W(z) = \frac{\mu}{2\pi}\frac{1}{z - z_0}$
速度势：$\phi = \frac{\mu}{2\pi}\frac{x - x_0}{(x - x_0)^2 + (y - y_0)^2}$
流函数：$\psi = -\frac{\mu}{2\pi}\frac{y - y_0}{(x - x_0)^2 + (y - y_0)^2}$
物理意义：
偶极子是点源和点汇的极限情况，表示流体从一个点流出并立即流入另一个点。偶极子产生的流场具有方向性，常用于模拟物体的绕流。
势流理论 偶极子 偶极矩
圆柱绕流
势流理论
$$W(z) = U_\infty\left(z + \frac{a^2}{z}\right) + \frac{\Gamma}{2\pi i}\ln\frac{z}{a}$$
圆柱绕流的复势函数
推导过程：
均匀流：$W_1(z) = U_\infty z$
偶极子：$W_2(z) = \frac{U_\infty a^2}{z}$
点涡：$W_3(z) = \frac{\Gamma}{2\pi i}\ln\frac{z}{a}$
叠加：$W(z) = W_1(z) + W_2(z) + W_3(z)$
复速度：$\frac{dW}{dz} = U_\infty(1 - \frac{a^2}{z^2}) + \frac{\Gamma}{2\pi i z}$
驻点位置：$\frac{dW}{dz} = 0$
物理意义：
圆柱绕流是势流理论的重要应用，通过叠加均匀流、偶极子和点涡得到。偶极子模拟圆柱的存在，点涡产生环量。驻点位置和压力分布可以通过复势函数计算。
势流理论 圆柱绕流 叠加原理
库塔条件
势流理论
$$\Gamma = 4\pi U_\infty a\sin\alpha$$
库塔条件，确定环量大小
推导过程：
圆柱绕流复速度：$\frac{dW}{dz} = U_\infty(1 - \frac{a^2}{z^2}) + \frac{\Gamma}{2\pi i z}$
在圆柱表面：$z = ae^{i\theta}$
切向速度：$v_\theta = -2U_\infty\sin\theta + \frac{\Gamma}{2\pi a}$
库塔条件：后驻点速度为零
后驻点：$\theta = \pi$
得到：$\Gamma = 4\pi U_\infty a\sin\alpha$
物理意义：
库塔条件确定了环量的大小，使得后驻点位于后缘。这个条件模拟了真实流动中的粘性效应，使得势流理论能够预测升力。库塔条件是连接势流理论与实际流动的桥梁。
势流理论 库塔条件 环量
升力定理
势流理论
$$L = \rho U_\infty \Gamma$$
库塔-茹科夫斯基升力定理
推导过程：
伯努利方程：$p + \frac{1}{2}\rho v^2 = p_\infty + \frac{1}{2}\rho U_\infty^2$
压力差：$\Delta p = p - p_\infty = \frac{1}{2}\rho(U_\infty^2 - v^2)$
切向速度：$v_\theta = -2U_\infty\sin\theta + \frac{\Gamma}{2\pi a}$
升力：$L = \int_0^{2\pi} -\Delta p a\sin\theta d\theta$
积分得到：$L = \rho U_\infty \Gamma$
升力方向：垂直于来流方向
物理意义：
升力定理是空气动力学的基本定理，表明升力与环量成正比。这个定理解释了机翼产生升力的原理，是飞机设计的基础。升力方向垂直于来流方向，指向环量方向。
势流理论 升力定理 库塔-茹科夫斯基
实际应用例题
势流理论
圆柱绕流升力计算
计算圆柱在均匀流中的升力
解题过程：
已知：$a = 0.1 \text{ m}, U_\infty = 10 \text{ m/s}, \alpha = 5° = 0.087 \text{ rad}$
库塔条件：$\Gamma = 4\pi U_\infty a\sin\alpha = 4\pi \times 10 \times 0.1 \times \sin(0.087) = 1.09 \text{ m}^2/\text{s}$
升力定理：$L = \rho U_\infty \Gamma = 1.225 \times 10 \times 1.09 = 13.4 \text{ N}$
升力系数：$C_L = \frac{L}{\frac{1}{2}\rho U_\infty^2 \times 2a} = \frac{13.4}{\frac{1}{2} \times 1.225 \times 10^2 \times 0.2} = 1.09$
理论值：$C_L = 2\pi\sin\alpha = 2\pi \times \sin(0.087) = 0.55$
差异原因：势流理论忽略粘性效应
结果分析：
圆柱绕流产生13.4N的升力，升力系数为1.09。理论值与实际值存在差异，这是因为势流理论忽略了粘性效应和边界层的影响。实际应用中需要考虑粘性修正。
