湍流基本概念

湍流具有随机性、旋涡性、耗散性和统计性等基本特征；课程中常把理论分析、实验研究和数值模拟作为三种主研究方法，再由 Reynolds 分解与雷诺应力把脉动流转成平均流描述。

湍流模型 \((k,\omega)\) \(R_{ij}\)
RANS方程推导
湍流理论
$$\frac{\partial \bar{u_i}}{\partial t} + \bar{u_j}\frac{\partial \bar{u_i}}{\partial x_j} = -\frac{1}{\rho}\frac{\partial \bar{p}}{\partial x_i} + \nu\frac{\partial^2 \bar{u_i}}{\partial x_j^2} - \frac{\partial \overline{u_i'u_j'}}{\partial x_j}$$
雷诺平均Navier-Stokes方程
推导过程：
瞬时Navier-Stokes方程：$\frac{\partial u_i}{\partial t} + u_j\frac{\partial u_i}{\partial x_j} = -\frac{1}{\rho}\frac{\partial p}{\partial x_i} + \nu\frac{\partial^2 u_i}{\partial x_j^2}$
雷诺分解：$u_i = \bar{u_i} + u_i', \quad p = \bar{p} + p'$
代入并展开：$\frac{\partial (\bar{u_i} + u_i')}{\partial t} + (\bar{u_j} + u_j')\frac{\partial (\bar{u_i} + u_i')}{\partial x_j} = -\frac{1}{\rho}\frac{\partial (\bar{p} + p')}{\partial x_i} + \nu\frac{\partial^2 (\bar{u_i} + u_i')}{\partial x_j^2}$
取时间平均：$\frac{\partial \bar{u_i}}{\partial t} + \bar{u_j}\frac{\partial \bar{u_i}}{\partial x_j} + \overline{u_j'\frac{\partial u_i'}{\partial x_j}} = -\frac{1}{\rho}\frac{\partial \bar{p}}{\partial x_i} + \nu\frac{\partial^2 \bar{u_i}}{\partial x_j^2}$
利用连续性方程：$\overline{u_j'\frac{\partial u_i'}{\partial x_j}} = \frac{\partial \overline{u_i'u_j'}}{\partial x_j}$
得到RANS方程：$\frac{\partial \bar{u_i}}{\partial t} + \bar{u_j}\frac{\partial \bar{u_i}}{\partial x_j} = -\frac{1}{\rho}\frac{\partial \bar{p}}{\partial x_i} + \nu\frac{\partial^2 \bar{u_i}}{\partial x_j^2} - \frac{\partial \overline{u_i'u_j'}}{\partial x_j}$
物理意义：
RANS方程描述了湍流平均流动的演化。雷诺应力项 $-\frac{\partial \overline{u_i'u_j'}}{\partial x_j}$ 表示湍流脉动对平均流的影响，这是湍流建模的核心问题。

雷诺应力张量
湍流理论
$$\tau_{ij} = -\rho \overline{u_i'u_j'} = \begin{pmatrix} \tau_{11} & \tau_{12} & \tau_{13} \\ \tau_{21} & \tau_{22} & \tau_{23} \\ \tau_{31} & \tau_{32} & \tau_{33} \end{pmatrix}$$
雷诺应力张量，对称二阶张量
推导过程：
雷诺应力定义：$\tau_{ij} = -\rho \overline{u_i'u_j'}$
对称性：$\tau_{ij} = \tau_{ji}$
湍动能：$k = \frac{1}{2}\overline{u_i'u_i'} = \frac{1}{2}(\tau_{11} + \tau_{22} + \tau_{33})/\rho$
各向同性部分：$\tau_{ij}^{iso} = \frac{2}{3}\rho k \delta_{ij}$
各向异性部分：$\tau_{ij}^{aniso} = \tau_{ij} - \tau_{ij}^{iso}$
涡粘性假设：$\tau_{ij}^{aniso} = 2\mu_t \bar{S}_{ij}$
物理意义：
雷诺应力张量描述了湍流脉动对平均流的应力贡献。对角元素表示法向应力，非对角元素表示剪切应力。涡粘性假设将各向异性部分与平均应变率联系起来。

2001 真题：湍流基本特征
原页短答
随机脉动性 + 旋涡性 + 强掺混性 + 耗散性 + 统计规律性
2001 年第 8 题可按随机脉动性、旋涡性、强掺混性、耗散性和统计规律性五个关键词依次展开。
对应页段：
随机脉动性：速度、压强等量围绕时均值不规则波动。
旋涡性：流场中充满多尺度旋涡结构。
强掺混性：动量、热量和质量交换显著增强。
耗散性：机械能最终经由粘性作用转化为内能。
统计规律性：瞬时运动复杂，但平均量与相关量仍可稳定描述。
若继续展开，可顺接 Reynolds 分解，并强调雷诺应力是脉动输运效应，不等于分子黏性应力本身。
原页说明：
原页把“统计规律性”和“耗散性”与前面的随机脉动、旋涡、强掺混并列列出，并顺势引到 Reynolds 分解与雷诺应力。
2001真题 湍流特征 雷诺应力
k-ε湍流模型
湍流理论
$$\frac{\partial k}{\partial t} + \bar{u_j}\frac{\partial k}{\partial x_j} = P_k - \varepsilon + \frac{\partial}{\partial x_j}\left[\left(\nu + \frac{\nu_t}{\sigma_k}\right)\frac{\partial k}{\partial x_j}\right]$$
湍动能k的输运方程
推导过程：
湍动能定义：$k = \frac{1}{2}\overline{u_i'u_i'}$
从脉动速度方程出发：$\frac{\partial u_i'}{\partial t} + \bar{u_j}\frac{\partial u_i'}{\partial x_j} + u_j'\frac{\partial \bar{u_i}}{\partial x_j} = -\frac{1}{\rho}\frac{\partial p'}{\partial x_i} + \nu\frac{\partial^2 u_i'}{\partial x_j^2}$
乘以$u_i'$并取平均：$\frac{\partial k}{\partial t} + \bar{u_j}\frac{\partial k}{\partial x_j} = -\overline{u_i'u_j'}\frac{\partial \bar{u_i}}{\partial x_j} - \varepsilon + \frac{\partial}{\partial x_j}\left[\left(\nu + \frac{\nu_t}{\sigma_k}\right)\frac{\partial k}{\partial x_j}\right]$
产生项：$P_k = -\overline{u_i'u_j'}\frac{\partial \bar{u_i}}{\partial x_j} = \nu_t\left(\frac{\partial \bar{u_i}}{\partial x_j} + \frac{\partial \bar{u_j}}{\partial x_i}\right)\frac{\partial \bar{u_i}}{\partial x_j}$
耗散项：$\varepsilon = \nu\overline{\frac{\partial u_i'}{\partial x_j}\frac{\partial u_i'}{\partial x_j}}$
涡粘性：$\nu_t = C_\mu\frac{k^2}{\varepsilon}$
物理意义：
k-ε模型通过求解湍动能k和耗散率ε的输运方程来封闭RANS方程。产生项表示平均流对湍流的能量输入，耗散项表示粘性耗散，扩散项表示湍流扩散。

耗散率ε方程
湍流理论
$$\frac{\partial \varepsilon}{\partial t} + \bar{u_j}\frac{\partial \varepsilon}{\partial x_j} = C_{1\varepsilon}\frac{\varepsilon}{k}P_k - C_{2\varepsilon}\frac{\varepsilon^2}{k} + \frac{\partial}{\partial x_j}\left[\left(\nu + \frac{\nu_t}{\sigma_\varepsilon}\right)\frac{\partial \varepsilon}{\partial x_j}\right]$$
耗散率ε的输运方程
推导过程：
耗散率定义：$\varepsilon = \nu\overline{\frac{\partial u_i'}{\partial x_j}\frac{\partial u_i'}{\partial x_j}}$
从脉动速度梯度方程出发
产生项：$C_{1\varepsilon}\frac{\varepsilon}{k}P_k$（与湍动能产生相关）
耗散项：$C_{2\varepsilon}\frac{\varepsilon^2}{k}$（ε的自耗散）
扩散项：$\frac{\partial}{\partial x_j}\left[\left(\nu + \frac{\nu_t}{\sigma_\varepsilon}\right)\frac{\partial \varepsilon}{\partial x_j}\right]$
模型常数：$C_{1\varepsilon} = 1.44, C_{2\varepsilon} = 1.92, \sigma_\varepsilon = 1.3$
物理意义：
耗散率方程描述了湍流能量耗散的演化。产生项表示平均流对耗散率的影响，耗散项表示ε的自耗散，扩散项表示耗散率的湍流扩散。ε方程与k方程耦合，形成完整的k-ε模型。

湍流能量级联
湍流理论
$$\varepsilon = \frac{u^3}{L}, \quad \eta = \left(\frac{\nu^3}{\varepsilon}\right)^{1/4}$$
能量级联和科尔莫戈罗夫尺度
推导过程：
能量级联假设：能量从大尺度向小尺度传递
特征速度：$u$（大尺度特征速度）
特征长度：$L$（积分尺度）
能量传递率：$\varepsilon = \frac{u^3}{L}$
科尔莫戈罗夫尺度：$\eta = \left(\frac{\nu^3}{\varepsilon}\right)^{1/4}$
泰勒微尺度：$\lambda = \sqrt{\frac{15\nu k}{\varepsilon}}$
物理意义：
湍流能量级联描述了能量从大尺度向小尺度的传递过程。科尔莫戈罗夫尺度是最小涡旋的特征尺度，在此尺度以下粘性耗散起主导作用。能量级联是湍流理论的核心概念之一。

实际应用例题
湍流理论
管道湍流流动分析
计算管道湍流流动的摩擦系数和压力损失
解题过程：
已知：$D = 0.1 \text{ m}, V = 2 \text{ m/s}, \rho = 1000 \text{ kg/m}^3, \mu = 0.001 \text{ Pa·s}$
雷诺数：$Re = \frac{\rho V D}{\mu} = \frac{1000 \times 2 \times 0.1}{0.001} = 200,000$
摩擦系数（Blasius公式）：$f = \frac{0.316}{Re^{1/4}} = \frac{0.316}{200,000^{1/4}} = 0.015$
压力损失：$\Delta p = f\frac{L}{D}\frac{\rho V^2}{2} = 0.015 \times \frac{10}{0.1} \times \frac{1000 \times 2^2}{2} = 3000 \text{ Pa}$
湍流强度：$I = \frac{\sqrt{k}}{V} = \frac{\sqrt{0.1}}{2} = 0.158$
涡粘性：$\nu_t = C_\mu\frac{k^2}{\varepsilon} = 0.09 \times \frac{0.1^2}{0.01} = 0.09 \text{ m}^2/\text{s}$
结果分析：
管道湍流流动中，雷诺数为200,000，处于充分发展湍流区。摩擦系数为0.015，压力损失为3kPa。湍流强度为15.8%，涡粘性约为分子粘性的90倍，体现了湍流的强烈混合特性。
