题目选路图：先判断它问什么

老师讲题时最怕学生一上来就翻公式。你先用一句话说出题目要你求什么：求流量，多半从连续方程走；求压强、速度和水头，多半从能量方程走；求支座力、壁面反力和射流冲击，多半从动量方程走；求模型和原型换算，就先找无量纲数；题里强调粘性、管流、边界层或湍流，就先判流态和边界条件。

1. 质量守恒

题干问流量、平均速度、截面积变化或密度变化，先写控制体进出量。不可压时常用 A1U1=A2U2，可压时别把密度丢掉。

2. 能量方程

题干有皮托管、文丘里、孔口、喷口、泵、压强差或水头损失，先问两点能不能连。再看是否有损失、外功和自由面压强。

3. 动量方程

题干问弯管受力、喷嘴反力、射流打板或流体对壁面的力，先画控制体、外法向和速度方向。力的正负比公式更容易错。

4. 量纲相似

题干出现模型、原型、比例尺、风洞、水槽或相似准则，先列变量量纲。自由面优先看 Fr，粘性阻力和管流优先看 Re。

5. 粘性、边界层、湍流

题干说层流、湍流、沿程损失、边界层厚度、摩阻或雷诺数，先判 Re 和壁面条件。粘性题常常输在流态判断。

写方程前先查三件事

“边界条件：固壁是不是无滑移，自由面是不是大气压，入口速度和出口压强有没有给。”

“控制体：进出口方向、外法向、压力作用面和重力方向先画出来。”

“单位：毫米、厘米、升每秒、千帕和水头米不要混。单位不齐，答案的数量级一定会跑。”

“检查完再去真题训练；卡在画面感时看自制动画；错在公式条件时回到公式回查。”

公式适用条件与错因回查表

选公式前先读题干，选公式后马上查条件。老师批卷时不是只看你等号写没写对，还看你有没有说明：这题为什么能用这个公式，题干别名是否归到同一入口，缺什么条件时不能直接用，边界条件有没有写清，方向、单位和真题重做入口有没有统一。

回真题重做
1. 题干条件

问流量、压强、受力、模型比例还是阻力损失。所求量没圈出来，公式就容易选偏。

2. 边界条件

固壁、自由面、入口出口、控制体外法向和速度方向，少一个都可能让正负号错。

3. 公式适用条件

定常、不可压、无旋、无损失、充分发展这些话，要从题干或图里找依据。

4. 单位/方向

水头米、帕、牛和立方米每秒要统一；动量题和相似题先写正方向和模型/原型比值。

答案不对时别急着重算，先查题干、边界、公式适用和单位方向，再决定重做哪类真题。

变量量纲表

$$[Q_i]=M^{a_i}L^{b_i}T^{c_i},\qquad [F]=MLT^{-2}$$

第一行不写公式，先写表。待求量、控制量、流体性质、几何量和外场量都列出来，逐个填 M、L、T 指数。

Buckingham π 定理

$$n\text{ 个变量},\ r\text{ 个基本量纲}\Rightarrow n-r\text{ 个 }\pi\text{ 群}$$

变量越多越要靠 π 定理降维。重复变量选错，n-r 这个数仍然对，但每个 π 群的物理意义会偏。

重复变量检查

$$\pi=Q\,\rho^\alpha U^\beta L^\gamma,\qquad [\pi]=1$$

把每个非重复变量乘上重复变量的幂，解指数方程。解完一定回代量纲，不能只看形式像不像。

Re：惯性力与黏性力

$$Re=\frac{\rho UL}{\mu}=\frac{UL}{\nu}$$

管流、绕流阻力、边界层厚度先看 Re。模型和原型若用同一流体且长度比很小，Re 相似常常最难满足。

Fr：惯性力与重力

$$Fr=\frac{U}{\sqrt{gL}}$$

自由面、船模、明渠、水跃、波浪题优先看 Fr。保持 Fr 相似会推出速度比等于长度比的平方根。

Eu：压力差与惯性压强

$$Eu=\frac{\Delta p}{\rho U^2}$$

题目问压差、压力系数或压降相似时看 Eu。写比例时要同时跟住密度比和速度比。

Ma：流速与声速

$$Ma=\frac{U}{a}$$

高速气体、可压缩、喷管和激波题看 Ma。低速不可压题可以说明 Ma 很小，不把它作为主相似准则。

St：非定常时间尺度

$$St=\frac{fL}{U}=\frac{L}{UT}$$

涡脱落、脉动流、周期模型实验看 St。它提醒你时间比不是随便取，要跟长度比和速度比配套。

力系数与阻力换算

$$C_F=\frac{F}{\rho U^2L^2},\qquad F\sim \rho U^2L^2$$

模型测到力以后，不是直接按长度平方放大。先确认控制的无量纲数相似，再用密度比、速度比和面积比换算。

模型/原型比例

$$\lambda_L=\frac{L_m}{L_p},\quad \lambda_U=\frac{U_m}{U_p},\quad \lambda_F=\lambda_\rho\lambda_U^2\lambda_L^2$$

先写“模型/原型”方向，别把比值倒过来。再由 Re、Fr、Ma 或 St 相似推出速度、时间、压差和力的比例。

边界条件与相似条件

$$\frac{x}{L},\ \frac{y}{L},\ \frac{u}{U},\ \frac{p}{\rho U^2}\text{ 的边界写成同形}$$

几何相似、运动相似、动力相似要一起说。固壁无滑移、自由面、入口速度型、出口压力这些条件也要按无量纲形式对应。

边界条件和单位检查

方程选对以后，还要看条件能不能支撑它。固壁、自由面、入口出口、控制体、流态和单位，是流体力学题最常见的扣分点。

回真题训练

固壁

粘性流动里固壁通常是无滑移；理想流体题里常用不可穿透。两个条件不要混着写。

自由面

开口水面常取大气压，速度有时可近似为零。能不能这样取，要看水箱面积和题干说明。

入口出口

入口速度型、出口压强、截面平均速度和外法向要对应。动量题尤其要把正方向写在图上。

单位

长度统一成米，压强统一成帕，流量统一成立方米每秒。水头米和压强帕之间要乘 rho g。

题面条件与推导场景

公式要落到题干条件里。船模、水槽、风洞、管流、喷管和脉动流这些场景看熟了，才能判断该走量纲相似、能量、动量，还是先处理粘性和边界层。

船模与明渠水槽

自由面波形和重力效应是主角，优先保持 Fr 相似。速度比、时间比通常由长度比开平方得到。

黏性阻力和边界层决定结果，优先看 Re。若同一流体做小模型，常要说明 Re 相似的速度要求是否现实。

风洞压差测量

模型测压孔、压力系数和阻力系数要写成 Eu 或 C_p、C_D，再用密度比和速度比换算到原型。

Eu压力系数

高速喷管与可压缩流

流速接近声速时，Ma 不能省。几何相似以外，还要说明入口总压、总温或压力比是否对应。

Ma可压缩

题里出现频率、周期或非定常相似，就把 St 写出来。时间比要由长度比和速度比推出。

复习顺序

学习顺序：题干条件 -> 边界条件 -> 公式适用条件 -> 单位/方向 -> 错因回查。前一步没分清，后面推导会越算越乱。

1. 题干条件

先圈已知量、所求量、图上方向和单位，不让条件散在题面里。

2. 边界条件

固壁、自由面、入口出口、控制体外法向和流态要能说清。

3. 公式适用条件

由题干条件选连续、能量、动量、相似或粘性路线，再说明为什么能用。

4. 单位/方向

统一量纲，标出正方向、法向、模型/原型比值和力的作用方向。

5. 错因回查

真题练判断，自制动画补画面，错题本只记录本题最容易再错的一句话。
