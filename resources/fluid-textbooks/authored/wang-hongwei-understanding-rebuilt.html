Teacher Route

老师读法：先把现象读准，再把公式写稳

版本标识：round241-human-teacher-upgrade-20260515。王洪伟这本适合用来把抽象概念放回生活现象里，但做题时仍要回到定义、方程适用条件、边界条件和真题训练。

第一遍：先看定义，读到能换句话说明就停

读第 1 章和每章开头时，先抓原书怎么定义流体、连续介质、控制体、流线、压强、粘性。每读完一个定义，停一下，用自己的话说清“它排除了什么误解”。说不清时不要急着往公式汇总翻。

第二遍：再看方程适用条件，读到边界条件能配齐再停

遇到连续方程、动量方程、Bernoulli、势流、粘性剪切和可压缩流时，先问是不是定常、不可压、无粘或一维平均流；再数未知量和边界条件。公式能不能用，往往就卡在这些条件上。

第三遍：最后做真题训练，把动画当检查图

静力学、皮托管、文丘里、射流冲击、管流损失、边界层和喷管题，先写题目给出的模型条件，再回到自制动画看图像是否同一件事。动画负责纠正“速度大压力小万能”“生活例子只是故事”这类误区，真题训练负责把第一行方程写出来。

读到哪里停

  • 读完定义后，能说清本节对象、适用范围和一个反例，再往下走。
  • 读完公式后，先标出忽略了什么作用、用了什么边界条件，再进例题。
  • 做完一组真题训练后，回查本页对应小节，看答案第一步是不是尊重原书条件。

补充说明

流体定义要抓剪切，而不是抓形状

水会流动，空气会扩散，这些都是现象。更有力的定义是：任意小的剪切应力只要持续存在，流体就会持续变形，不能像固体那样靠一个静止剪切变形把它平衡掉。

生活例子不是插曲，而是理解入口

水滴、气泡、胶管出流、覆杯实验这些例子把抽象物性变成可观察现象。知识点全库适合做考点，王洪伟这本适合把考点重新点亮。

相关问题

  • 定义流体时强调剪切应力与持续变形。
  • 连续介质题写出尺度分离。
  • 表面张力题要联系曲率和界面能。

从剪切响应定义流体
  1. 固体可在小剪切变形后产生弹性剪应力并静止。

表面张力到曲率压差的说明
  1. 自由表面可看成具有张力的界面膜。
  2. 弯曲界面两侧压力差必须由表面张力的法向合力平衡。

  1. 气体压力、密度和温度由状态方程联系。
  2. 当流动速度或外力足以造成明显密度变化时，不能再用不可压假设。

关键推导

从剪切响应定义流体

1.1 · 原书页码线索 PDF p.16-20

  1. 固体可在小剪切变形后产生弹性剪应力并静止。
  2. 流体若存在持续剪切应力，会持续发生剪切变形。
  3. 因此流体与固体的本质差别不在于能否变形，而在于能否靠静止变形维持剪应力。
  4. 这一定义自然导向粘性和速度梯度。

条件限制

  • 不宜把“没有固定形状”当唯一答案。

表面张力到曲率压差的说明

1.2.2 · 原书页码线索 PDF p.25-28

  1. 自由表面可看成具有张力的界面膜。
  2. 弯曲界面两侧压力差必须由表面张力的法向合力平衡。
  3. 两个主曲率半径共同决定 \(\Delta p=\sigma(1/R_1+1/R_2)\)。
  4. 水滴、气泡和毛细现象都可从这个平衡出发解释。

条件限制

  • 平面界面曲率为零时没有 Laplace 曲率压差。

状态方程和压缩性进入流体力学

1.2.3-1.2.4 · 原书页码线索 PDF p.27-30

  1. 气体压力、密度和温度由状态方程联系。
  2. 当流动速度或外力足以造成明显密度变化时，不能再用不可压假设。
  3. 低速空气常可近似不可压，是因为密度变化量级很小。
  4. 高速、强旋转、大高度差或显著传热都会把压缩性带回问题。

条件限制

  • 不可压近似不是液体专属，也不是永远适用于气体。

压强是各向同性法向应力

对流项不是玄学

  4. 反对称部分对应刚体式转动，并与涡量相连。

条件限制

  • 涡量是微团角速度的二倍，不宜把二者混用。

\[\frac{D}{Dt}=\frac{\partial}{\partial t}+\mathbf{u}\cdot\nabla\]

不可压条件

\[\nabla\cdot\mathbf{u}=0\]

微团体积变化率为零。

速度梯度分解

\[\nabla\mathbf{u}=\mathbf{D}+\mathbf{W}\]

变形率张量加转动张量。

加速度展开

\[\mathbf{a}=\frac{\partial\mathbf{u}}{\partial t}+(\mathbf{u}\cdot\nabla)\mathbf{u}\]

流体质点加速度的欧拉表达。

涡量与角速度

\[\boldsymbol{\omega}=2\boldsymbol{\Omega}\]

微团刚体转动角速度与涡量关系。

雷诺输运定理

\[\frac{d}{dt}\int_{CV}\rho b\,dV+\oint_{CS}\rho b(\mathbf{u}\cdot\mathbf{n})\,dA\]

系统量转控制体量的核心结构。

补充说明

无粘和无旋不能互换

无粘是力学假设，无旋是运动学性质。无粘流可以有涡量，无旋流也可能只在某些区域成立。很多错题都来自把这两个词混成一个词。

势流的工程地位来自“主结构”

对于高 Re 外流，主体外流常接近无旋，势流能给出压力分布、升力趋势和附加质量；阻力和分离则要靠边界层、尾迹和实验修正。

补充说明

边界层是粘性和惯性的折中

壁面附近粘性不可忽略，外层惯性主导。边界层理论正是把这两个区域分开处理，让看似不可解的 N-S 方程变成可分析的问题。

分离是近壁动量不够了

逆压梯度要求流体逆着压力升高方向前进，近壁流体速度小、动量弱，容易被迫减速到零甚至倒流，于是流线离开壁面。

相关问题

  • 边界层厚度要区分名义厚度、位移厚度和动量厚度。
  • 分离题必须说明逆压梯度和近壁动量不足。

  5. Eu 比较压力变化和惯性尺度。

喷流推力的控制体动量说明

9.4-9.6 · 原书页码线索 PDF p.262-266

  1. 围绕气球、水火箭或发动机取控制体。
  2. 喷出流体带走动量通量。
  3. 控制体外力与动量通量差平衡。
  4. 主要推力量级可写成 \(F\sim\dot{m}U\)，若出口压力不等于环境压还要加压力项。

条件限制

  • 推力与喷出介质类型无关，关键是动量通量。

总压、动压和冲力的统一解释

9.7-9.9 · 原书页码线索 PDF p.266-272

  1. 流体被无损滞止时，动能转为压力能。
  2. 不可压近似下总压等于静压加动压。
  3. 物体使流体减速或转向时，动量变化表现为冲力。
  4. Bernoulli 和动量方程是在不同角度描述同一个减速过程。

条件限制

  • 高速可压或有损流动中总压关系要修正。

动压

\[q=\frac{1}{2}\rho U^2\]

速度对应的动能压力尺度。

总压近似

\[p_0=p+\frac{1}{2}\rho U^2\]

不可压无损滞止关系。

射流推力尺度

\[F\sim \dot{m}U\]

动量通量决定推力尺度。

弯曲流线法向压强梯度

\[\frac{\partial p}{\partial n}=\rho\frac{U^2}{R}\]

弯曲流线需要法向压力梯度。

喷管推力

\[F=\dot{m}(U_e-U_0)+(p_e-p_a)A_e\]

喷气和水火箭推力的完整结构。

毛细尺度

\[Bo=\frac{\rho gL^2}{\sigma}\]

重力与表面张力相对重要性。

压力系数

\[C_p=\frac{p-p_\infty}{\tfrac{1}{2}\rho U_\infty^2}\]

把局部压力变化写成动压尺度。

毛细数

\[Ca=\frac{\mu U}{\sigma}\]

粘性作用与表面张力作用的比值。
