涡量定义

涡量理论
ω = ∇ × v
表示流体微团的转动强度，是描述旋转运动的重要物理量。

推导线索

  • 从速度场 v 出发，对速度取旋度。
  • ∇ × v 给出局部转动方向和强度。
  • 涡量方向沿转动轴，大小等于角速度的两倍。

物理意义

当 ω = 0 时为无旋流动；当局部流体微团存在自转时，涡量能直接描述旋转轴和旋转强度。

旋度无旋流速度场

涡量方程

涡量理论
Dω/Dt = (ω · ∇)v + ν∇²ω
描述涡量在流体中的演化，核心是拉伸项与粘性扩散项。

推导线索

  • 对 Navier-Stokes 方程取旋度。
  • (ω · ∇)v 表示涡线拉伸与扭曲。
  • ν∇²ω 表示粘性对涡量的扩散和耗散。

物理意义

拉伸会改变涡量强度和方向，扩散会让涡量随粘性作用逐渐衰减，是理解湍流结构的重要入口。

输运方程拉伸项粘性扩散

流体静力学基本方程

流体静力学
∇p = ρg
静止流体中，压强梯度与单位体积重力相平衡。

推导线索

  • 取静止流体微元分析受力。
  • 体积力为 ρg dV，表面力为 -∇p dV。
  • 平衡条件给出 ∇p = ρg。

物理意义

它说明压强空间变化来自重力场，是水利、海洋和大气静力近似中最基础的平衡关系。

压强梯度静力平衡重力场

静水压强分布

流体静力学
p = p₀ + ρgh
不可压缩流体中，压强随深度 h 线性增加。

推导线索

  • 从 dp/dz = -ρg 出发。
  • ρ 为常数时直接积分。
  • 令 h = z₀ - z，得到 p = p₀ + ρgh。

物理意义

压强与深度成正比，斜率为 ρg。计算坝面压力、浮力和海水压力时都从这个关系开始。

不可压缩深度静水压力

伯努利方程

流体动力学
p/(ρg) + v²/(2g) + z = H
沿流线机械能守恒，压强水头、速度水头和位置水头之和保持不变。

推导线索

  • 沿流线积分理想流体欧拉方程。
  • 要求定常、不可压、无粘，并沿同一流线。
  • 积分后得到单位重量机械能守恒。

物理意义

速度增大时压强或高度会相应降低。管流、孔口出流、文丘里管和机翼近似分析都常用它。

能量守恒沿流线总水头

动量方程

流体动力学
F = ∬ ρv(v · n)dA + ∬ p n dA
控制体受力与动量通量、压力作用相关，是工程受力计算的核心工具。

推导线索

  • 从牛顿第二定律写控制体动量平衡。
  • 把流入流出的动量通量计入边界积分。
  • 再加上压力、重力等外力项。

物理意义

用来计算射流冲击、弯管受力、喷嘴反推力等问题。关键是选好控制体和法向方向。

控制体动量通量受力分析
