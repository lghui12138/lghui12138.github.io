# 🌊 流体力学学习平台 - 模块化海洋版

## 📖 项目简介

这是一个专为中国海洋大学803流体力学专业设计的现代化学习平台，采用模块化架构和海洋主题设计，提供全面的流体力学学习体验。

## ✨ 主要特性

### 🎨 海洋主题设计
- **动态波浪背景** - 三层波浪动画效果
- **游动的海洋生物** - 鱼群动画装饰
- **海洋渐变色彩** - 蓝色系主题配色
- **活泼的交互效果** - 悬停动画和过渡效果

### 🏗️ 模块化架构
- **独立功能模块** - 每个功能都是独立的HTML文件
- **统一设计风格** - 所有模块保持一致的海洋主题
- **易于维护** - 模块化设计便于开发和维护
- **可扩展性** - 新功能可以轻松添加为新模块

## 📚 功能模块

### 1. 🏠 主界面 (`index-modular.html`)

- **抖音视频支持** - 抖音视频链接导入
- **视频分类管理** - 按类型和难度分类

- **学习建议** - AI提供的学习建议
- **计划管理** - 创建、编辑、删除计划

### 7. 🤖 AI助手模块 (`modules/ai-assistant-module.html`)
- **智能问答系统** - 基于知识库的问答

## 🚀 快速开始

### 环境要求
- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 本地HTTP服务器（推荐使用Python）

### 启动步骤

1. **启动本地服务器**
   ```bash
   python -m http.server 8000
   ```

2. **访问主界面**
   ```
   http://localhost:8000/index-modular.html
   ```

3. **登录系统**
   - 管理员账户：`admin` / `Admin123!`
   - 学生账户：`student` / `Student123!`

4. **探索功能模块**
   - 点击各个模块卡片进入相应功能
   - 或直接访问原版完整界面
